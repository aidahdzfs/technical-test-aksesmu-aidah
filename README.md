# Product Management System (Laravel 11 + Tailwind CSS v4)

Aplikasi manajemen produk sederhana dengan fitur CRUD yang dibangun menggunakan arsitektur MVC serta menerapkan standar *Clean Code* dan *Best Practices* Laravel.

   cd NAMA_REPO_ANDA
2. **Instal Dependensi PHP & Node.js**
    ```bash
    composer install**
    npm install
3. **Setup Enviroment File**
    ```bash
    copy .env.example .env

    ```bash
    php artisan migrate --seed
